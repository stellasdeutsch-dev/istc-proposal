# ISTC — Digital Proposal & Website Redesign

An independent, pre-hire proposal for the **International Science and Technology Center** (istc.int), created by **Shyngys Narseyit** as an applicant for the PR & Media Intern role.

Two Apple-inspired, fully responsive, animated, **trilingual (EN / RU / KK)** deliverables.

## 1 · The Proposal — `/`
A single-page pitch built around the candidate's real edge: the platforms ISTC is missing (TikTok, Telegram, Threads) are exactly the ones he has already grown to **145,000 followers**. Includes the current-state audit, a phased 60-day plan, measurable commitments, and a link to the live redesign.

## 2 · The Redesign — `/redesign/`
A modern, **multi-page** concept redesign of the ISTC website:

| Page | Path |
|------|------|
| Home | `/redesign/` |
| About | `/redesign/about.html` |
| What We Do | `/redesign/what-we-do.html` |
| Projects | `/redesign/projects.html` |
| News | `/redesign/news.html` |
| Contact | `/redesign/contact.html` |

**Modern UX/UI patterns:** sticky translucent nav with active-page state, full-screen mobile menu, an animated `<canvas>` particle-network hero, scroll-reveal animations, count-up stats, filterable project grid, timeline, contact form, back-to-top, and a persistent EN/RU/KK language switcher.

## Technical notes
- Shared design system in `redesign/assets/` — one `styles.css`, one `main.js`, one `i18n.js` (all translations).
- No build step, no framework, no external assets. `prefers-reduced-motion` respected throughout.
- **Independent concept**, not an official ISTC website. Figures are drawn from a public digital audit (July 2026); ISTC content is used only for demonstration.
