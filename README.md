# ISTC — Digital Proposal & Website Redesign

An independent, pre-hire proposal for the **International Science and Technology Center** (istc.int), created by **Shyngys Narseyit** as an applicant for the PR & Media Intern role.

Two Apple-inspired, fully responsive, animated static sites:

| Page | What it is | Path |
|------|------------|------|
| **The Proposal** | A 60-day communications plan — where ISTC stands online today, the phased plan, and measurable deliverables. | [`/`](./index.html) |
| **The Redesign** | A modern concept redesign of the ISTC homepage — same mission, a first impression that matches it. | [`/redesign/`](./redesign/index.html) |

## Notes
- Both pages are single self-contained HTML files: no build step, no dependencies, no external assets.
- Animations use CSS transitions, `IntersectionObserver`, and a lightweight `<canvas>` particle network. `prefers-reduced-motion` is respected.
- This is an **independent concept**, not an official ISTC website. Figures are drawn from a public digital audit (July 2026); ISTC content is used only for demonstration.
